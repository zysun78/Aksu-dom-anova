  塔里木河地下水DOM的Two-way ANOVA分析工具
  
═══════════════════════════════════════════════════

📖 使用方法:
1. 双击 index.html 文件
2. 在浏览器中自动打开(推荐使用Chrome/Edge)
3. 无需网络连接,完全离线可用

📊 主要功能:
- 基于TDS和ORP中位数的双因素方差分析
- 8个DOM参数的统计检验结果
- 交互式图表(可缩放、悬停查看详情)
- 内置完整R和Python分析代码
- 导出CSV数据表和高清PNG图片

💡 常见问题:
Q: 打开后显示乱码?
A: 使用Chrome/Edge浏览器打开

Q: 图表不显示?
A: 确保浏览器允许运行JavaScript

Q: 如何导出结果?
A: 点击页面上的"Export"按钮
